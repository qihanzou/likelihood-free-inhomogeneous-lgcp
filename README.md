Codes for Deep Simulation-Based Inference (DSBI) for Inhomogeneous Log-Gaussian Cox Processes.

The main_codes folder contains the main code for this project:
1. NN_DSBI_sim.py contains the functions for the DSBI model used in the simulation component.
2. NN_DSBI_app.py contains the functions for the DSBI model used in the application component.
3. simulation.R is used for the simulation study.
4. application.R is used for the application to the gorilla data.
5. application_kfold.R implements the k-fold–like algorithm.
