# Likelihood-Free Methods for Univariate Inhomogeneous LGCPs

Code for likelihood-free inference methods for univariate inhomogeneous Log-Gaussian Cox processes.

This repository contains scripts and functions for fitting and analysing univariate inhomogeneous LGCP models, where the intensity is modeled as

$$
\log \Lambda(s) = \mu(s) + Y(s),
$$

with $\mu(s)$ representing the inhomogeneous mean trend and $Y(s)$ representing the latent Gaussian spatial field.
